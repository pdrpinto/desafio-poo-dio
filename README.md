## Repositorio dedicado a desafio pratico pelo bootcamp Java AWS pela DIO.

Neste pequeno projeto, foram postos em pratica conceitos sobre heranca, encapsulamento e polimorfismo. 
Aqui, pode-se registrar desenvolvedores e cadastra-los para consumir conteudos (Curso ou Mentoria).
Alem disso, foi implementado ranking geral para acompanhar a evolucao de cada desenvolvedor, alem
da possibilidade de deixar comentarios em foruns existentes em cada um dos conteudos oferecidos.
